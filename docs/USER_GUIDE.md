## 🚀 快速上手

### 启动应用

1. 双击 SnapMark 图标启动应用
2. 启动后，应用会自动最小化到系统托盘（屏幕右下角）
3. 您会看到鼠标指针周围出现一个炫酷的光圈效果

### 进入绘图模式

**方法一**：使用快捷键（默认 `Alt + `` ` ，即 Alt 加键盘左上角的波浪键）

**方法二**：右键点击系统托盘图标，选择"绘制模式"

进入绘图模式后，屏幕顶部会出现工具栏。

### 基本操作

1. **绘制标注**：选择工具栏中的画笔工具，在屏幕上拖动鼠标绘制
2. **选择颜色**：点击工具栏中的颜色按钮，选择喜欢的颜色
3. **清除标注**：使用快捷键 `Alt + Q` 清除所有标注
4. **退出绘图模式**：按下 `Escape` 键或再次按下切换快捷键

## 🎮 三大模式详解

### 空闲模式 (Idle)

空闲模式是 SnapMark 的默认模式。在这个模式下，鼠标可以自由穿透 SnapMark 窗口，您可以正常操作其他应用程序，同时享受鼠标光圈和点击特效带来的视觉效果。

**进入方式**：

- 启动应用后自动进入
- 在绘图模式或截图模式下按下 `Escape` 键
- 使用切换快捷键（默认 `Alt + `` ` ）

**功能特点**：

- ✅ 鼠标光圈效果正常显示
- ✅ 点击特效正常触发
- ✅ 键盘回显正常显示
- ✅ 鼠标可穿透窗口，不影响操作其他应用

**注意事项**：

- 在空闲模式下，鼠标点击会穿透 SnapMark 窗口，直接作用于下方的应用程序
- 如果需要在 SnapMark 上绘图，必须先切换到绘图模式

### 绘图模式 (Draw)

绘图模式是 SnapMark 的核心模式。在这个模式下，您可以在屏幕上自由绘制标注，使用各种绘图工具。

**进入方式**：

- 使用切换快捷键（默认 `Alt + `` ` ）
- 右键点击系统托盘图标，选择"绘制模式"

**工具栏**：

工具栏位于屏幕顶部，包含以下工具：

| 工具       | 图标描述 | 功能                   |
| -------- | ---- | -------------------- |
| **选择工具** | 鼠标箭头 | 选择、移动、编辑已绘制的图形       |
| **画笔工具** | 画笔   | 自由绘制线条               |
| **直线工具** | 直线   | 绘制直线                 |
| **箭头工具** | 箭头   | 绘制带箭头的线条             |
| **矩形工具** | 矩形   | 绘制矩形（按住 Shift 绘制正方形） |
| **椭圆工具** | 椭圆   | 绘制椭圆（按住 Shift 绘制圆形）  |
| **文本工具** | 字母 T | 添加文字标注               |
| **橡皮擦**  | 橡皮擦  | 擦除标注内容               |

**颜色选择器**：

工具栏左侧有颜色选择器，点击可以选择绘制颜色。包含预设颜色和自定义颜色选项。

**粗细调节**：

工具栏右侧有线条粗细调节滑块，可以调整画笔和形状的粗细。

**侧面板**：

将鼠标移动到屏幕左右边缘，会弹出侧面板（素材库），可以添加和管理常用的图形素材。

**自动隐藏**：

- 工具栏和侧面板会在鼠标离开后 2 秒自动隐藏
- 将鼠标移动到屏幕顶部边缘，工具栏会自动弹出
- 将鼠标移动到屏幕左右边缘，侧面板会自动弹出
- 正在绘制时（鼠标按键按下），不会触发自动隐藏

**操作技巧**：

- **撤销**：`Ctrl + Z`
- **重做**：`Ctrl + Shift + Z`
- **删除选中对象**：`Delete` 或 `Backspace`
- **复制**：`Ctrl + C`
- **粘贴**：`Ctrl + V`
- **全选**：`Ctrl + A`

### 截图模式 (Screenshot)

截图模式允许您选择屏幕区域进行截图，并自动保存到剪贴板。

**进入方式**：

- 使用快捷键 `Alt + S`（可自定义）
- 右键点击系统托盘图标，选择"截图"

**选区形状**：

截图模式支持三种选区形状：

| 形状       | 操作方式    | 说明         |
| -------- | ------- | ---------- |
| **矩形选区** | 拖动鼠标    | 绘制矩形区域（默认） |
| **圆形选区** | 按住特定键拖动 | 绘制圆形区域     |
| **自由选区** | 按住特定键拖动 | 自由绘制选区边界   |

**操作步骤**：

1. 进入截图模式后，屏幕会变暗，显示半透明遮罩，顶部会出现选区形状切换工具栏
2. 选择选区形状（矩形、圆形、套索），在屏幕上拖动鼠标选择区域
3. 松开鼠标后，自动完成截图并保存到剪贴板
4. 您可以在其他应用中粘贴截图
5. 按下 `Escape` 键退出截图模式，返回绘图模式

## ⚙️ 设置界面说明

### 打开设置界面

**方法一**：右键点击系统托盘图标，选择"偏好设置"

**方法二**：使用命令（需先进入绘图模式）

设置界面包含五个标签页：鼠标、键盘、绘图、通用、关于。

### 鼠标设置

鼠标设置页面包含两部分：点击特效和鼠标光圈。

#### 点击特效

| 设置项        | 说明                             |
| ---------- | ------------------------------ |
| **启用点击特效** | 开关，控制是否在鼠标点击时显示特效              |
| **特效类型**   | 下拉菜单，选择特效样式：波纹、烟花、螺旋、圆形描边、方形描边 |
| **缩放**     | 滑块，调节特效大小（0.1 - 5.0）           |
| **速度**     | 滑块，调节特效动画速度（0.1 - 5.0）         |
| **主颜色**    | 颜色选择器，设置特效的主色调                 |
| **副颜色**    | 颜色选择器，设置特效的副色调                 |

#### 鼠标光圈

| 设置项        | 说明                                                     |
| ---------- | ------------------------------------------------------ |
| **启用光圈**   | 开关，控制是否显示鼠标光圈                                          |
| **启用光圈动画** | 开关，控制光圈是否有动态效果                                         |
| **光圈大小**   | 滑块，调节光圈尺寸（0.1 - 1.0）                                   |
| **光圈样式**   | 下拉菜单，选择光圈视觉效果：霓虹、金色、极光、火焰、冰霜、彩虹、阴影、闪烁、萤火虫、波纹、旋转器、漩涡、射线 |

### 键盘设置

键盘设置页面用于配置键盘回显功能。

| 设置项        | 说明                           |
| ---------- | ---------------------------- |
| **启用键盘回显** | 开关，控制是否在屏幕上显示按键              |
| **启用点击回显** | 开关，控制是否显示鼠标按键                |
| **缩放**     | 数字输入，调节按键显示大小（0.5 - 3.0）     |
| **X 轴偏移**  | 滑块，调节按键显示位置的水平偏移（-200 到 200） |
| **Y 轴偏移**  | 滑块，调节按键显示位置的垂直偏移（-200 到 200） |
| **字体颜色**   | 颜色选择器，设置按键文字颜色               |
| **背景颜色**   | 颜色选择器，设置按键背景颜色               |

### 绘图设置

绘图设置页面用于配置绘图相关的快捷键。

| 设置项          | 说明                                |
| ------------ | --------------------------------- |
| **切换模式快捷键**  | 自定义进入/退出绘图模式的快捷键，默认为 `Alt + `` `  |
| **显示工具栏快捷键** | 自定义显示/隐藏工具栏的快捷键                   |
| **清除标注**     | 固定快捷键 `Alt + Q`，用于清除所有标注          |
| **快捷键帮助**    | 固定快捷键 `Shift + /`，显示快捷键帮助         |

### 通用设置

通用设置页面包含应用的全局设置。

| 设置项      | 说明                  |

| **开机自启** | 开关，控制应用是否在系统启动时自动运行 |

## ❓ 常见问题

### Q1: 启动后看不到窗口怎么办？

A: SnapMark 启动后会自动最小化到系统托盘。您可以在屏幕右下角找到托盘图标，右键点击查看菜单选项。

### Q2: 如何进入绘图模式？

A: 使用快捷键 `Alt + `` ` （默认），或右键点击托盘图标选择"绘制模式"。

### Q3: 绘图时工具栏挡住了内容怎么办？

A: 工具栏会在鼠标离开后自动隐藏。您也可以将鼠标移动到屏幕中间区域，工具栏会自动隐藏。

### Q4: 鼠标光圈太亮/太暗怎么办？

A: 右键点击托盘图标进入设置界面，在"鼠标设置"中调整光圈样式和大小。

### Q5: 键盘回显位置不对怎么办？

A: 在设置界面的"键盘设置"中调整 X 轴偏移和 Y 轴偏移。

### Q6: 如何清除屏幕上的所有标注？

A: 使用快捷键 `Alt + Q`，或在工具栏中选择橡皮擦工具手动擦除。

### Q7: 截图后图片保存在哪里？

A: 截图会自动保存到剪贴板，您可以粘贴到其他应用（如画图、Word、聊天软件等）中。

### Q8: 如何退出应用？

A: 右键点击系统托盘图标，选择"退出"。

## 💡 使用注意事项

1. **首次使用建议**：
   - 先进入设置界面熟悉各项功能
   - 根据个人喜好调整鼠标光圈样式和点击特效
2. **教学演示技巧**：
   - 使用醒目的颜色标注重点内容
   - 配合键盘回显功能展示快捷键操作
   - 使用箭头工具指向关键位置
3. **性能注意事项**：
   - 过于复杂的特效可能会占用较多系统资源
   - 如果发现卡顿，可以降低特效的缩放和速度
4. **多显示器支持**：
   - SnapMark 支持多显示器环境
   - 在截图模式下，可以跨显示器选择区域
5. **快捷键冲突**：
   - 如果默认快捷键与其他应用冲突，可以在设置中自定义
   - 建议选择不常用的组合键作为切换快捷键
