## 🚀 사용법

### 기본 조작

1. **앱 실행**: 앱이 시스템 트레이로 최소화됩니다
2. **설정 열기**: 트레이 아이콘 → 설정
3. **기능 활성화**:
   - "마우스" 탭에서 클릭 효과 활성화
   - "키보드" 탭에서 키보드 에코 활성화
   - "그리기" 탭에서 단축키 설정
4. **사용 시작**: 단축키를 눌러 화면에 그리기 시작

### 🎯 기타 기능

- ✅ **크로스 플랫폼**: Windows, macOS, Linux 지원
- ✅ **투명 창**: 그리기 시 화면 콘텐츠를 가리지 않습니다
- ✅ **단축키**: 사용자 정의 단축키로 빠르게 기능 전환
- ✅ **사용자 정의 스타일**: 색상, 크기, 속도 등의 매개변수 조정 가능
- ✅ **자동 시작**: 시스템 부팅 시 자동 시작 설정 가능
- ✅ **시스템 트레이**: 시스템 트레이로 최소화하여 언제든 접근 가능

## 🛠️ 개발

### 개발 환경 요구 사항

- Node.js >= 18
- Bun >= 1.0
- Rust >= 1.70
- OS별 요구 사항:
  - **Windows**: WebView2
  - **macOS**: Xcode Command Line Tools
  - **Linux**: webkit2gtk, libgtk-3-dev

### 프로젝트 구조

```
penio/
├── src/                    # 프론트엔드 소스 코드
│   ├── components/         # React 컴포넌트
│   ├── hooks/              # 커스텀 Hooks
│   ├── i18n/               # 국제화
│   ├── pages/              # 페이지
│   ├── store/              # 상태 관리
│   └── utils/              # 유틸리티
├── src-tauri/              # Tauri 백엔드
│   ├── src/                # Rust 소스 코드
│   ├── icons/              # 앱 아이콘
│   └── capabilities/       # 권한 설정
├── docs/                   # 문서 및 데모 영상
└── public/                 # 정적 자원
```
