## 🚀 Verwendung

### Grundlegende Bedienung

1. **App starten**: Die App minimiert sich in den Systemtray
2. **Einstellungen öffnen**: Tray-Symbol klicken → Einstellungen
3. **Funktionen aktivieren**:
   - Klickeffekte im Tab "Maus" aktivieren
   - Tastatur-Echo im Tab "Tastatur" aktivieren
   - Shortcuts im Tab "Zeichnung" konfigurieren
4. **Loslegen**: Drücken Sie den Shortcut, um auf dem Bildschirm zu zeichnen

### 🎯 Weitere Funktionen

- ✅ **Plattformübergreifend**: Unterstützt Windows, macOS, Linux
- ✅ **Transparentes Fenster**: Zeichnen verdeckt den Bildschirminhalt nicht
- ✅ **Tastenkombinationen**: Anpassbare Shortcuts für schnelles Umschalten der Funktionen
- ✅ **Anpassbare Stile**: Farben, Größen, Geschwindigkeiten und andere Parameter anpassen
- ✅ **Autostart**: Optionaler automatischer Start beim Systemstart
- ✅ **Systemtray**: Minimierung in den Systemtray für einfachen Zugriff

## 🛠️ Entwicklung

### Entwicklungsanforderungen

- Node.js >= 18
- Bun >= 1.0
- Rust >= 1.70
- OS-spezifische Anforderungen:
  - **Windows**: WebView2
  - **macOS**: Xcode Command Line Tools
  - **Linux**: webkit2gtk, libgtk-3-dev

### Projektstruktur

```
penio/
├── src/                    # Frontend-Quellcode
│   ├── components/         # React-Komponenten
│   ├── hooks/              # Benutzerdefinierte Hooks
│   ├── i18n/               # Internationalisierung
│   ├── pages/              # Seiten
│   ├── store/              # Zustandsverwaltung
│   └── utils/              # Hilfsfunktionen
├── src-tauri/              # Tauri-Backend
│   ├── src/                # Rust-Quellcode
│   ├── icons/              # App-Icons
│   └── capabilities/       # Berechtigungskonfiguration
├── docs/                   # Dokumentation und Demo-Videos
└── public/                 # Statische Ressourcen
```
